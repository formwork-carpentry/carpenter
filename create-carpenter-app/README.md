# create-carpenter-app

Professional scaffolding package for Carpenter projects.

## Installation

```bash
npm install create-carpenter-app
```

## Usage

```bash
npx create-carpenter-app --help
```

## Scripts

| Script | Command |
| --- | --- |
| build | tsc -p tsconfig.json |
| test | vitest run |
| typecheck | tsc --noEmit -p tsconfig.json |
