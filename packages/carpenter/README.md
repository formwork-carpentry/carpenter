# carpenter-cli

Carpenter CLI — scaffold and manage Carpenter framework applications

## Installation

```bash
npm install carpenter-cli
```

## Usage

```bash
npx carpenter --help
```

## Scripts

| Script | Command |
| --- | --- |
| build | tsc -p tsconfig.json |
| typecheck | tsc --noEmit -p tsconfig.json |
