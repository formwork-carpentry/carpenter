# Carpenter

Scaffold new Carpenter projects with the right folder structure and formwork packages.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/formwork-carpentry/carpenter.git
cd carpenter

# Install with bun
bun install

# Create a new project
bunx create-carpenter-app my-app
```

## Packages

- **carpenter** — CLI wrapper package
- **@formwork/cli** — Full CLI implementation (generators, migrations, serve, build)
- **create-carpenter-app** — Interactive project scaffolder

## Formwork Packages

The building blocks live in the [formworks](https://github.com/formwork-carpentry/formworks) repo.
